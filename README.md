# -DC-MOD-BOT

Moderation Commands:

/purge - Bulk delete messages (1-100)
/kick - Kick users with reason
/ban - Ban users with reason
/mute - Timeout users with duration parsing (10m, 1h, 2d)
/warn - Issue warnings with tracking

Invite Tracking:

Automatic tracking on member join
/invites - Check user's invite count
/topinvites - Top 10 leaderboard

Giveaway System:

/giveaway_start - Start with duration, min invites, optional role
/giveaway_end - Manual end
Interactive button to join
Automatic winner selection
Requirement validation (invites + role)

Additional Features:

Auto-role for new members
/ping - Latency check
/help - Command list
